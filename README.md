# 🎰 GifRoulette

GifRoulette is a fun and lightweight web application that randomly displays GIFs based on a roulette-style selection.

## 🚀 Features

- 🎲 Random GIF selection
- 🔄 Refresh / spin functionality
- ⚡ Fast and simple interface
- 🎯 Minimalist design

## 🛠️ Tech Stack

- Frontend: HTML / CSS / JavaScript  
- (Optional) Backend: Node.js / Express (if applicable)  
- API: Giphy API (or any GIF provider)

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/sdr-dev-96/gifroulette.git
cd gifroulette
```

Install dependencies (if applicable):
```bash
npm install
```

Run the project:
```bash
npm start
```

## 🔧 Configuration
If you are using the Giphy API, create a .env file:
```env
API_KEY=your_giphy_api_key
```

## 🎮 Usage

```bash
node main.js "keyword"
```
